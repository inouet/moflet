Moflet
======

Moflet is a Simple and Fast PHP framework.


Installation
------

### Setup composer

    $ curl -s https://getcomposer.org/installer | php


### Create composer.json 

    {
        "require": {
            "moflet/core": "dev-master"
         }
    }

### Install Moflet

    $ php composer.phar install

### Setup Project

    $ php vendor/moflet/core/bin/mof-init
